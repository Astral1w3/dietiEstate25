# dietiEstate25

Consiglio di utilzzare docker per avviare il backend

# Prerequisiti

Prima di iniziare, assicurati di avere installato i seguenti strumenti:

    JDK 21 o superiore.

    Maven 3.x.

    PostgreSQL: Un'istanza di PostgreSQL in esecuzione.

    Un IDE a tua scelta (es. IntelliJ IDEA, Eclipse, VS Code con le estensioni Java).
