# Dieti Estates 2025 Monorepo

Questa repository contiene l'intero progetto:

- `dietiestates2025-back`: backend Spring Boot
- `dietiestates2025-front`: frontend React
- `Database`: dump e documentazione DB
- `Docker`: note Docker

## Deploy

- Backend workflow: `.github/workflows/backend-azure-deploy.yml`
- Frontend workflow: `.github/workflows/frontend-azure-static-web-app.yml`

Entrambi i workflow sono configurati per il branch `main`.
